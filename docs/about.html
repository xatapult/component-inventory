About component-inventory

The component-inventory website is a completely static website (so no search but also no cookies…). It is built with XML technology, using XProc and XSLT to generate all the HTML pages. Some texts were generated using an AI. Parts of the software were built with technology developed within my company Xatapult. Generating the full website from its XML sources takes less than 10 seconds.

As far as I know, all additional information on this website, like datasheets, is public and not copyright protected. If I have accidentally violated anyone’s intellectual property, please let me know.

The source code and data for this website is hosted on Github. Everything is published under the GNU General Public License, giving you complete freedom to copy, share, reuse, or modify it to your heart’s content. However, you will not be able to run it: it requires several additional, not always published, components and XML catalogs are used for internal referencing. See the repository’s README file for more information.

The site is published using GitHub Pages.

 

If you find a bug or want to comment on the site’s contents, please use the GitHub issues mechanism and raise an issue.

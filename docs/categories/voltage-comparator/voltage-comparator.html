 Category: Voltage comparators 

A voltage comparator is an electronic circuit designed to compare two input voltages and switch its output to a defined high or low state depending on which input is greater. Unlike an op-amp, which is optimized for linear amplification, a comparator is built for fast, clean transitions and typically operates in saturation, making it ideal for turning analog signals into digital logic levels. When the non-inverting input rises above the inverting input, the output snaps to its high state; when it falls below, the output switches low. This sharp switching behavior makes comparators essential in applications like threshold detection, zero-crossing detection, waveform shaping, and many types of control and timing circuits.
